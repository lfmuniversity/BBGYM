# BBGYM

Pacchetto pronto per GitHub Pages.

## Come pubblicare

1. Crea una nuova repository su GitHub, per esempio `bbgym`.
2. Carica TUTTI i file di questa cartella nella root della repository.
3. Vai su **Settings → Pages**.
4. In **Build and deployment**, scegli:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salva.
6. Apri il link GitHub Pages da Safari su iPhone.
7. Fai **Condividi → Aggiungi a schermata Home**.

L’icona iPhone è `apple-touch-icon.png`.
I dati vengono salvati nel browser del dispositivo.
